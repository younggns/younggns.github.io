Overview

I took SI 539 - Design of Complex Web Sites, and designed and implemented a webpage for personal portfolio. Also, I paired up with another student and made a team blog website powered by Heroku and Flask.

 

Details

One of the main features of the personal portfolio website was a WordCloud to describe my interests. I used D3.js library to create word clouds and modified them as responsive svg objects.

 

For the teamblog, I tried to visualize blog posts into force-directed graph - each node represents each blog post and edges between nodes describe the relationship between blog posts. For example, if two posts were about movie review, they are connected to each other, but a blog post about technology cannot be connected with these nodes. I implemented front-end view of the website(drawing graphs by using D3.js library), and my teammate managed back-end side.

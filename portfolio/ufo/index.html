Overview

I took SI 650 - Information Retrieval course and paticipated in the project for developing a software related to information retrieval. Our group developed a software for searching records about UFOs based on NUFORC website.

 

Process

1. Initial Crawling and Data Scraping] As a first step, we've crawled the website to grab UFO records from NUFORC website, and cleaned this dataset.

2. Building Search Engine] In order to create inverted indexes from the dataset and search engine, we've imported Whoosh as a library for building search engine.

3. Retrieving Relevant Set] For retrieving relevant set of results, we adopted several methodologies such as PageRank, query expansion, and cosine similariy.

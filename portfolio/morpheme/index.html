Overview

This project was for analyzing Korean text and sorting them by co-occurrence of morphemes in the text. By the result of analysis, users can easily see the relatedness of words - if word A appeared with word B a lot, while it didn't appear with word C, then we can assume that word A is more related to word B than word C.

 

Details

In this project, due to the inherent characteristic of Korean morphemes and grammar, it’s hard to directly apply text analysis methodology in English text. So we devised a system based on vector analysis. Each element in a N-dimension vector represents the number of advents in N-th sentence. Therefore, if we have 5 sentences and the word “is” appears in 1st, 3rd and 4th sentence, it’s vector representation is (1, 0, 1, 1, 0). In this way, if we calculate cosine value from the inner product of two vectors, that value can represent correlation of two words. If two words have higher cosine value than usual pairs of words, then we can put these words nearby.

 

Some of the words in a text are regarded as significant than others, based on the position in a paragraph. Since important words tent to appear in the first or the last sentence of a paragraph. We put more weight on those words to calculate “importance factor”. In visualizations, we can use “correlation value” to determine the relative position of a morpheme and “importance factor” to determine the size of a morpheme.
